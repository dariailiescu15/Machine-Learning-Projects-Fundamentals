# Machine Learning — Projects & Fundamentals

Acest repository conține o colecție de proiecte practice care parcurg treptat conceptele esențiale din Machine Learning, de la pregătirea datelor până la modele avansate de clasificare și clustering. Fiecare notebook explorează un subiect diferit, folosind seturi de date reale și tehnici standard din domeniu.

Proiectele sunt implementate în Python, folosind biblioteci precum pandas, numpy, matplotlib, seaborn, scikit-learn.

## 🔹 1. Preprocesarea datelor
Notebook‑urile inițiale acoperă:

încărcarea și curățarea datelor

tratarea valorilor lipsă

encoding pentru variabile categorice

normalizare și standardizare

împărțirea datelor în train/test

Aceste etape sunt fundamentale pentru orice proiect ML.

## 🔹 2. Regresie
Proiectele includ:

regresie liniară

evaluarea performanței (MAE, MSE, RMSE, R²)

vizualizarea relațiilor dintre variabile

interpretarea coeficienților

Seturi de date precum house_1.csv sunt folosite pentru a modela prețuri sau valori continue.

## 🔹 3. Clasificare
Notebook‑urile dedicate clasificării explorează:

logistic regression

k‑Nearest Neighbors

decision trees

evaluare prin accuracy, precision, recall, F1-score

matrice de confuzie și curbe ROC

Fișiere precum clasificare_1.csv sunt folosite pentru exerciții practice.

## 🔹 4. Ensemble Methods
Proiectele includ:

Random Forest

Gradient Boosting

Bagging & Boosting

comparații între modele

Fișierele ensemble_1.csv și ensemble_1.txt sunt folosite pentru experimente.

## 🔹 5. Clustering
Ultimele proiecte explorează tehnici nesupervizate:

K-Means

determinarea numărului optim de clustere (Elbow, Silhouette)

vizualizarea clusterelor

aplicarea clustering‑ului pe seturi de date reale (ex: vietnam_housing_dataset.csv)

Acesta marchează tranziția ta către ML nesupervizat.

## Obiectivul repository‑ului
Scopul acestui repo este să documenteze progresul în învățarea Machine Learning‑ului, prin proiecte practice care acoperă:

preprocesare

modele supervizate

modele nesupervizate

evaluare și interpretare

vizualizare și analiză
